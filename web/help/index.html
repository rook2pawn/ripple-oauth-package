Help
  1. First register a 3rd party app - your app will receive a client secret
  2. You will need to fill a redirect URI value
  3. we will supply a widget that will post back the authorization code to your server
  4. That server will then submit its client secret along with one of your users authorization code
  5. we will give back an access token good for the permissions the user authorized
  6. user has capability to revoke access (through /manage) on a granular basis (e.g. funds revocation)
